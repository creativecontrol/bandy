Class: BandyClient

BandyClient()

bandy Client is the mobile web end of the bandy interactive music platform. It presents information about the performance and provides several games that participants can play to have an affect on the generated music. Players log into the system anonymously for ease of use. The bandy Controller aggregates all connected Players' data to generate musical events.

Class: BandyController

BandyController()

bandy Controller is the backend of the bandy interactive music platform. It aggregates all player data and feeds it into an algorithm for the generation of MIDI events. It also allows for defining of system wide variables and facilitates editing of information for players attending the performances. These MIDI events can be sent to MIDI devices connected to the Controller computer. Generated MIDI events are also stored to the database for access by the bandy Performers.
